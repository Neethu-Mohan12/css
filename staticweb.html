Build static websites
with an easy-to-use framework

Middleman is a static site generator using all the shortcuts and tools in
modern web development

Installation

Install the gem and run middleman init my_projectto create
a new Middleman site from the command line.

Customize Your Site

Check out the directory cd my_project. Middleman works right out
of the box. Start reading the docs to find out what each directory is
doing and how to customize your site.

Build and Deploy

Running one command, bundle exec middleman build, exports
the site in a production-ready format. Choose from open source
deployement solutions to get your site live.


Open-Source Community


Middleman benefits from an active community. Whether you need a question
answered or want to contribute to open-source features, our community
page will lead you in the right direction.
